## Overview
StudyBuddy is an AI-powered study partner matching application designed specifically for National Tsinghua University students. The platform helps students find compatible study partners based on shared courses, preferred study locations, and available time slots using a sophisticated machine learning algorithm.

## ‚ú® Features
- **Smart Matching Algorithm**: Uses cosine similarity to find the most compatible study partners
- **Course-Based Matching**: Match with students taking similar courses
- **Flexible Preferences**: Set study locations and time preferences
- **Real-Time Search**: Dynamic course search with instant results
- **Modern UI**: Engaging animations powered by GSAP
- **Secure Authentication**: Session-based user management
